# LLM Analysis

This repository contains the public research package for a model-level study of moral framing bias, sycophancy, model-specific bias fingerprints, and inference-time debiasing methods.

The main package is:

- [`moral-bias-debias-study/`](moral-bias-debias-study/)

## What Is Included

- Reproducible experiment code for moral framing and sycophancy evaluations.
- Inference-time debiasing methods including standard prompting, structured debate, critique-revise, counterfactual consistency voting, constitutional critique, truth-first prompting, and frame-invariant rationales.
- Analysis scripts for model bias fingerprints, method responsiveness, and deeper regularity analysis.
- Public summary reports and CSV tables for completed pilot/full runs.

## What Is Excluded

Raw API logs, API keys, model checkpoints, and very large raw trial-level tables are intentionally excluded. The uploaded reports contain the important aggregate results needed for writing, presentation, and follow-up experiments.
